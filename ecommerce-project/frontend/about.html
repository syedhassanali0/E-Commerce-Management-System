About this project

ShopEase is an E-Commerce Management System built as the final semester project for Open Source Software Development (OSSD – Y9). It lets customers browse products, search and filter by category, manage a shopping cart, and place orders, while administrators manage products, categories, and orders from a dedicated dashboard.

How it works

The frontend (HTML, CSS, JavaScript) talks to a FastAPI backend over a REST API. The backend stores all data in a PostgreSQL database (hosted on Supabase) using SQLAlchemy. Passwords are hashed with bcrypt and users stay logged in with JWT tokens.

Main features

  • User registration and secure login
  • Product listing with search and category filter
  • Shopping cart with quantity management
  • Order placement and status tracking
  • Admin dashboard with live sales analytics
  • Full CRUD operations for products, categories, and orders
